# Portfólio 

Bem-vindo ao meu projeto de Portfolio construído ao longo do curso de HTML e CSS da Alura!

![image](https://github.com/Hekth/portfolio/assets/151862970/caf0e51f-2573-46db-b4ab-9ce2faf532fb)

## Descrição do projeto

O objetivo deste projeto é apresentar meu trabalho, meus projetos desenvolvidos, tecnologias com experiência, meus contatos e um pouco sobre mim.

Ainda está em fase inicial então não está completo.

O projeto possui 3 páginas: Home, Sobre mim, Currículo.

### Home

Consta com uma breve descrição chamativa, uma foto minha e algumas opções de contatos. É a página inicial da aplicação.

### Sobre mim

Consta algumas informações pertinentes sobre a minha pessoa, principalmente em relação ao âmbito profissional.

### Currículo

Possui informações sobre meu currículo.

## Ferramentas utilizadas:
* HTML
* CSS
* Flexbox
* MediaQueries
